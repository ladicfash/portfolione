## 📂 Repository Structure

/  
  index.html        # Main portfolio landing page
  styles.css        # CSS for site layout and theme
  app.js            # JavaScript interactivity
  assets/  
    images/         # Portfolio images/posters
    pdf/            # Cheat sheets, certificates, reports
    pptx/           # Slide decks, if any
    archive/        # Deprecated or unused assets

## 🖼️ Adding New Projects

1. Place new project images in assets/images/.
2. Place certificates, cheat sheets, or reports in assets/pdf/.
3. Reference them in index.html using relative links.
4. (Optional) For large decks, include download links.
